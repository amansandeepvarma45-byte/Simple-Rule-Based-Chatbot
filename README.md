# Simple Rule-Based Chatbot

A simple chatbot developed in Java using pattern matching and predefined rules.

## Features

* Greeting responses (Hi, Hello, Hey)
* Help command support
* Small talk interactions
* Cybersecurity knowledge base
* Conversation history logging
* Interactive console-based chat

## Technologies Used

* Java
* ArrayList
* Scanner
* Rule-Based Pattern Matching

## Sample Questions

* Hi
* Help
* How are you?
* What is Cyber Security?
* What is Phishing?
* What is Firewall?
* What is Kali Linux?

## How to Run

1. Compile the program:
   javac RuleBasedChatbot.java

2. Run the program:
   java RuleBasedChatbot
